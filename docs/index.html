Timeline Line

A tiny library to draw a line around DOM elements.

Simple demo

Custom demo

It's beta for now and only works on vertical timelines.

Add Timeline-line

Add the timeline script to your html

<script src="https://unpkg.com/timeline-line"></script>

You should also be able to grab from NPM

npm i timeline-line

Quick setup

To get going quickly you can just use the 'fromClass' function to setup.

new Timeline(document.getElementById('svg'), Timeline.fromClass('block', 'right'));

Styling

Timeline-line does not apply any styles to the svg path so don't forget to add those to your css. See examples of HTML and CSS setup in the examples below.
